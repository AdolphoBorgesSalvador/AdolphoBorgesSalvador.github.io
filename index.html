Projetos

Planejamento de Estoque (ABC/XYZ + Cobertura)

Pipeline em Python que lê planilhas PSI e Mapa (variáveis de ambiente via .env), executa cálculo de médias 12/6/3 meses, mediana, classificação ABC/XYZ, soma de importações previstas e projeta saldo mês a mês (Jul→Out/25) para estimar cobertura (MOS) com/sem importação e sinalizar status (ex.: “Necessita compra Outubro”). Suporta integração com PostgreSQL (SQLAlchemy) e gera tabelas finais para decisão.

Código no GitHub

Forecast de Demanda com LSTM (ABC/XYZ)

Pipeline em Python que consolida históricos (planilhas SAP), trata e agrega por mês, classifica materiais por ABC/XYZ e treina redes LSTM (Keras) para prever a demanda mensal por canal e por material (foco em classe AZ). Usa sliding window, MinMaxScaler e compara previsões com o real via MAPE, gerando tabelas e gráficos para tomada de decisão.

Código no GitHub

Extração de Dados de PDFs

Automação com PyMuPDF para estruturar dados (CPF/RG/endereço) em JSON; integrações para camada base.
